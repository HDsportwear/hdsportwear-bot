# HD Sportwear Telegram Bot

Версия 1.2 официального Telegram-бота HD Sportwear.

## Railway variables

Обязательные:

- `BOT_TOKEN` — токен BotFather.
- `ADMIN_CHAT_ID` — Telegram ID администратора.
- `PAYMENT_GROUP_ID` — ID закрытой группы для квитанций и подтверждения оплат.
- `SUPPORT_GROUP_ID` — ID группы для вопросов клиентов, статусов, обменов и возвратов. Можно указать ту же группу, что и `PAYMENT_GROUP_ID`.

Дополнительные:

- `COMPANY_NAME` — `HD Sportwear`
- `SITE_URL` — `https://hdsportwear.shop`
- `DELIVERY_URL` — `https://hdsportwear.shop/delivery`
- `MANAGER_URL` — `https://t.me/Danil_HDsportwear`
- `PAYMENT_RECIPIENT` — ФИО получателя ФОП
- `PAYMENT_IBAN` — IBAN
- `PAYMENT_BANK` — название банка
- `PAYMENT_TAX_ID` — ИНН получателя
- `DISCOUNT_UAH` — `100`

## Как узнать ID группы

1. Добавьте бота в закрытую группу.
2. Выдайте боту право отправлять сообщения.
3. В группе отправьте `/chatid`.
4. Полученное отрицательное число укажите в `PAYMENT_GROUP_ID` или `SUPPORT_GROUP_ID`.
